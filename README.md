# Task Management System

A full-stack task management application built with ASP.NET Core and Angular.

## Tech Stack

Backend:
- ASP.NET Core Web API
- Entity Framework Core
- JWT Authentication

Frontend:
- Angular
- TypeScript
- Bootstrap

Database:
- SQL Server

Cloud:
- Azure App Service
- Azure Static Web Apps
- Azure SQL Database
